SCOOT

  • R, Shiny, Leaflet, API
  • Transit Mapping

I developed SCOOT which is an interactive real-time transportation mapping software. SCOOT queries the location of the user (with their permission), and then uses that information to map the most direct route to their destination given the selected mode. SCOOT uses the Google Maps API and the Translink API to get real-time transportation data.

Flood Anomaly Detection System

  • R, SQL
  • LOESS Regression

I developed a real-time anomaly detection system for river flow gauges across North America for BGC Engineering. The system is implemented using a Robust Mean LOESS method in R and Python. This system was able to classify anomalies with an F1 score of 83% on a dataset of over 33 million observations.

2-Day Flood Forecasting System

  • Python, sklearn
  • Random Forest Regression

I developed a 2-day flood forecasting system using Python and Scikit Learn for BGC Engineering. This system uses historic flow, precipitation, and spatial data to train a Random Forest model. Sommelier

  • R, Shiny

Sommelier is a Shiny web app that helps consumers find a wine to match their tastes and budget. Sommelier accesses the Winemag database of over 150,000 wine reviews and finds the highest rated wines at the lowest price, for a given grape variety and origin. Value is determined by the magnitude of the linear regression residual.
